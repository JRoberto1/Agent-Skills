## O que há de novo na v3

**web.md:**
- 7 direções de design com DNA completo (resolve o "AI slop")
- Imagens sem API: picsum seeds + pravatar + CSS puro
- Copywriting embutido: benefícios vs funcionalidades
- Componente de navbar mobile pronto e acessível
- CSS base com tipografia fluida e scroll animations

**app.md:**
- Imagens sem API integradas
- Touch targets obrigatórios (44×44px)
- Acessibilidade mobile detalhada

**content.md:**
- Copywriting por segmento integrado
- Tom por tipo de conteúdo

**debug.md:**
- Diagnóstico de visual genérico adicionado
- Referência rápida expandida

## Regras Fixas (sempre aplicadas)

- SEO: máxima performance no Google
- Mobile-first: sempre
- Ícones: SVG (Lucide) — nunca emojis
- Imagens: catálogo verificado — nunca API em tempo real para listagens
- Gradiente roxo/azul: NUNCA
- Fonte Inter genérica: NUNCA
- Lorem Ipsum: NUNCA
